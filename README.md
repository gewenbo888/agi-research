# AGI Research

A curated reading room for research on artificial general intelligence.

[![Live Demo](https://img.shields.io/badge/demo-agi--research--seven.vercel.app-3c5688)](https://agi-research-seven.vercel.app)

## Overview

A Next.js site that collects and presents research pointing towards AGI — papers, labs, approaches, and open questions — in a readable, browsable format suitable for both students and practitioners.

## Design & Tech

- **Stack:** Next.js, React, Tailwind CSS
- **Deployment:** Vercel

## Getting Started

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
